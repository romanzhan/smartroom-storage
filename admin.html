Справка: как работают расчёты

1. Стоимость хранения

В калькуляторе две вкладки: Boxes (коробки) и Furniture (мебель).

  • Коробки: стоимость = сумма (цена × кол-во) по каждой позиции. Цена за 4 недели.
  • Мебель: стоимость = цена выбранного юнита хранения. Цена за 4 недели.

Скидка за срок применяется к итоговой цене хранения (см. секцию «Скидки за срок»). Скидка не применяется к rolling-плану (помесячная оплата без фиксированного срока).

2. Скидки за срок хранения

Таблица скидок отсортирована по убыванию мин. месяцев. Применяется первый подходящий уровень:

если срок >= 12 мес → 45%
если срок >= 9 мес  → 35%
если срок >= 6 мес  → 25%
если срок >= 3 мес  → 15%
если срок >= 2 мес  → 5%
иначе               → 0%

Значения настраиваются в секции «Скидки за срок» выше.

3. Стоимость перевозки (Collection Fee)

Рассчитывается только для режима «Home collection». Для «Drop-off» стоимость перевозки = 0.

3.1 Определение грузчиков

если объём > autoUpgradeThreshold → 2 грузчика
иначе                             → 1 грузчик

3.2 Малые заказы (объём < smallJobThreshold)

Используются фиксированные значения времени:

погрузка     = smallJobLoadingTime (фикс.)
разгрузка    = smallJobUnloadingTime (фикс.)
время_пути   = расстояние × smallJobTravelPerMile + smallJobTravelDelay
базовая_цена = max(общее_время / 60 × ставка, smallJobMinPrice)

3.3 Обычные заказы (объём >= smallJobThreshold)

эффективность   = baseTimePerCubicMeter × (efficiencyFactor / (movers + 0.2))
множитель_этажа = 1 + этаж × (floorMultiplierNoLift | floorMultiplierWithLift)
  (для ground floor множитель = 1.0)

погрузка   = эффективность × объём × множитель_этажа
разгрузка  = эффективность × объём × 1.0  (склад = 1 этаж)
время_пути = расстояние × travelTimePerMile + fixedDelayMinutes

базовая_цена = общее_время / 60 × ставка_грузчика

3.4 Наценка за перегруз

если объём > overloadThreshold  → цена × overloadHeavyMultiplier
если объём > vanCapacity        → цена × overloadLightMultiplier
иначе                           → без наценки

3.5 Наценки по дате/времени

Каждая наценка — процент от текущей цены, добавляемый сверху:

срочность (< urgencyDaysThreshold дней)  → + urgencySurcharge
праздник                                  → + holidaySurcharge (заменяет выходные)
выходные (сб/вс)                          → + weekendSurcharge
2-часовое окно                            → + twoHourWindowSurcharge

3.6 Доп. услуги и НДС

доп_услуги     = сумма (цена_услуги × кол-во) для per-item услуг
итого_до_ндс   = цена_перевозки + доп_услуги
ндс            = итого_до_ндс × ставка_ндс  (если включён и applyToCollection)
итого          = итого_до_ндс + ндс

4. Расстояние

Расстояние от адреса клиента до склада рассчитывается через Google Maps Distance Matrix API (driving mode). Если API недоступен — используется значение по умолчанию (normalJobDefaultMiles или smallJobDefaultMiles).

Координаты склада задаются в секции «Общие настройки».

5. Итоговый счёт клиента

первый_платёж = стоимость_хранения + страховка + стоимость_перевозки
ежемесячно    = стоимость_хранения + страховка

Страховка выбирается клиентом (фиксированные тарифы, не настраиваются в админке).
